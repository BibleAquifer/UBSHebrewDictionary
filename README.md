# _UBS Dictionary of Biblical Hebrew_ Aquifer Resource

This repository (`UBSHebrewDictionary`) contains Aquifer resource data with resource-level metadata, article-level metadata, and content files in JSON and Markdown formats.

## Structure

The repository is organized by language codes, with each language folder containing subfolders for JSON and Markdown content.

## Documentation

For more information about the Aquifer platform, data, and metadata formats, visit the [Aquifer Documentation repository](https://github.com/BibleAquifer/UBSHebrewDictionary).
